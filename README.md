# Assignment 4 — NLOS Imaging

This repository contains our MATLAB implementation for Assignment 4 on non-line-of-sight (NLOS) imaging.

It includes:
- transient visualization (`x-t` and `y-t` slices),
- naive voxel-wise backprojection,
- Laplacian / Laplacian-of-Gaussian (LoG) filtering,
- confocal vs. non-confocal comparison on `bunny`,
- attenuation compensation experiments,
- phasor / Morlet temporal filtering.

---

## Setup

1. Open MATLAB.
2. Set the working directory to the root folder of the project.
3. Add all subfolders to the MATLAB path:

```matlab
addpath(genpath(pwd))
```

4. Place the provided `.mat` datasets either:
   - in the root project folder, or
   - inside a subfolder called `datasets/`.

Expected dataset names:
- `Z_d=0.5_l=[1x1]_s=[256x256].mat`
- `usaf_d=0.5_l=[1x1]_s=[256x256].mat`
- `bunny_d=0.5_l=[1x1]_s=[256x256].mat`
- `bunny_d=0.5_c=[256x256].mat`

---

## Folder Structure

- `datasets/`  
  Input `.mat` datasets.

- `results_modular/`  
  Output figures and summaries generated by the scripts.

- `main_run.m`  
  Main entry point for the assignment.

- `default_cfg.m`  
  Central configuration file with dataset paths, output folders, voxel resolutions, wall stride and phasor parameters.

- `section1_transients.m`  
  Generates `x-t` and `y-t` transient slices.

- `section2_backprojection_z_usaf.m`  
  Runs naive backprojection on `Z` and `usaf` with different voxel resolutions and wall subsampling.

- `section3_bunny_compare.m`  
  Compares non-confocal and confocal reconstruction for `bunny`.

- `section4_attenuation.m` / `section4_attenuation_v2.m`  
  Attenuation compensation experiments.

- `section5_phasor_sweep.m`  
  Morlet / phasor sweep for the confocal `bunny` dataset.

---

## How to Run

The project is executed from `main_run.m`.

Open `main_run.m` and set the section you want to run by changing the flag:

```matlab
cfg.section = "transients";
```

Available sections are:
- `"transients"`
- `"z_usaf"`
- `"bunny"`
- `"attenuation"`
- `"phasor"`

Then run:

```matlab
main_run
```

---

## Recommended Order

1. `transients`  
   Check that dataset loading is correct and inspect the transient structure.

2. `z_usaf`  
   Validate the backprojection pipeline and compare voxel resolution / wall stride.

3. `bunny`  
   Compare non-confocal and confocal geometry.

4. `attenuation`  
   Evaluate whether conservative attenuation compensation changes the result.

5. `phasor`  
   Compare the phasor result against the confocal LoG baseline.

## Output

Generated figures and summaries are saved under `results_modular/`, for example:
- `results_modular/section_1_transients/`
- `results_modular/section_2_backprojection/`
- `results_modular/section_3_bunny/`
- `results_modular/section_4_attenuation/`
- `results_modular/section_5_phasor/`

Depending on the script version, some folders may have slightly different names such as:
- `section_4_attenuation_v2`
- `baseline_confocal_log`
- `run_XX_lambda_..._sigma_...`
